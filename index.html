Your Doc

In the "YourDoc" project, I took the lead in designing and developing a smart doctor appointment booking system. The core objective of this system was to provide a convenient platform for patients to schedule appointments with doctors based on their availability, eliminating the need for in-person clinic visits. This not only enhanced the patient experience but also streamlined the appointment scheduling process for healthcare providers. To ensure a seamless and paperless consultation process, I engineered a patient profile system. This system allowed doctors to upload medical reports and share them with patients or other medical professionals. The backend was developed using Node.js, while the frontend was built with React.js, ensuring a user-friendly and responsive interface. Furthermore, I implemented rigorous unit testing and integration testing to ensure the reliability and performance of the system. To facilitate the deployment and continuous improvement of the project, a streamlined Continuous Integration/Continuous Deployment (CI/CD) pipeline was established. This project aimed to revolutionize the way patients interact with healthcare services by making doctor appointments more accessible and efficient.

Data Pipeline

In the data pipeline project, I designed and implemented an efficient AWS-based data pipeline to automate the conversion of JSON to CSV format, facilitating streamlined data analysis in Amazon QuickSight. Leveraging Amazon S3 for storage and AWS Lambda for event-driven triggers, the pipeline ensured seamless and real-time processing of data. The project also incorporated AWS Glue for data integration, providing optimized data flow and storage. Terraform was utilized for infrastructure as code, ensuring scalability and maintainability. The end-to-end process, from data ingestion to transformation and analysis, adhered to best practices in data management, enabling robust insights for decision-making.
